## Tasks

Analyze the topic library and provide:

### 1. 本周发布计划 (Weekly Plan)
Based on priority and stage, suggest which 3 topics to publish this week. For each:
- Which topic
- Which platform first
- Best time to post
- One preparation tip

### 2. 内容缺口分析 (Gap Analysis)
Identify what's missing:
- Which themes are thin (few topics)?
- Which platforms are underused?
- What content formats are missing?
- Suggest 3-5 new topic ideas to fill gaps

### 3. 系列内容机会 (Series Opportunities)
Find topics that can become series:
- Which single topics can expand into 3-5 part series?
- What series name and structure?

### 4. 平台适配建议 (Platform Adaptation)
For each topic marked "Ready", suggest:
- How to adapt for secondary platforms
- Title adjustments per platform
- Format changes needed

### 5. 涨粉策略 (Growth Strategy)
Based on current stage distribution:
- What to focus on now to reach next follower milestone
- Which content types drive follower growth
- Quick wins vs long-term plays

## Output Format

Use the configured language. Output as markdown with clear sections. Be specific and actionable — not generic advice.
