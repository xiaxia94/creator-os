# Creator OS — Codex 适配器

> **推荐方式：** 本项目已内置 Codex Skill，位于 `.codex/skills/creator-os/SKILL.md`。
> 
> 用户只需：
> 1. 克隆/下载项目
> 2. 用 Codex 打开项目目录
> 3. 直接说"用 Creator OS"或输入 `/creator-os`
> 
> Codex 会自动发现并加载 Skill，无需任何额外配置。

---

## 手动使用（备选）

如果你的 Codex 版本不支持自动发现 Skill，可以手动加载：

1. 打开 Codex
2. 将本文件（`adapters/codex/system-prompt.md`）的内容作为系统提示词
3. 按提示词中的流程操作

详细指令请参阅 `.codex/skills/creator-os/SKILL.md`。
