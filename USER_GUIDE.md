## 🚀 5 分钟快速开始

### 第一步：安装 Creator OS

#### 方法 A：Claude Code 用户（推荐）

1. **下载项目**
   - 在 GitHub 页面点击绿色 `Code` 按钮 → `Download ZIP`
   - 解压到任意位置（如桌面）

2. **在 Claude Code 中使用**
   - 打开 Claude Code
   - 直接开始使用（Claude Code 会自动识别 skill）

#### 方法 B：Codex 用户

1. 下载并解压项目
2. 用 Codex 打开项目文件夹
3. 在 Codex 中开始使用

#### 方法 C：Cursor 用户

1. 下载并解压项目
2. 用 Cursor 打开项目文件夹
3. 在 Cursor 中开始使用

#### 方法 D：其他 AI 助手

1. 下载并解压项目
2. 将 `skill.md` 或 `adapters/` 中的适配器文件加载到你的 AI 助手
3. 开始使用

---

### 第二步：首次使用 — 告诉 AI 你是谁

**只需说一次**，Creator OS 会记住你的身份：

```
我是一个 AI 创作者。
主要平台：小红书和 YouTube。
目标：建立个人影响力。
```

或者更详细：

```
我是 AI 创作者。
主要平台：小红书、公众号、YouTube。
目标：分享 AI 工具和创作经验。
当前阶段：刚开始做，粉丝不到 1000。
```

**AI 会自动：**
- ✅ 创建你的创作者画像
- ✅ 初始化选题库
- ✅ 准备好接收你的灵感

---

### 第三步：日常使用 — 粘贴灵感

**不需要任何格式**，直接粘贴你的想法：

#### 示例 1：零散想法

```
AI 周报自动化帮我每周省 2 小时
见海外客户最大的坑：没确认时区
为什么大多数人学 AI 都放弃了：没有即时 ROI
```

#### 示例 2：完整备忘录

```
今天的思考：
* AI 自动化周报真的省时间，从 2 小时变成 15 分钟
* 和海外客户开会一定要确认时区，上次差点误事
* 很多人学 AI 坚持不下去，因为看不到即时效果
```

#### 示例 3：灵感 + 画像（首次使用）

```
我是 AI 创作者，主做小红书和 YouTube。

今天的想法：
* AI 周报自动化：从 2 小时到 15 分钟
* 海外客户开会的 5 个坑
* 为什么 90% 的人学 AI 坚持不下去
```

### 第四步：查看结果

AI 处理完后，你会看到：

```
✅ 本次处理完成！

📥 新增 3 个选题：
1. 「AI 周报自动化：从 2 小时到 15 分钟」
   爆款 75 / 人设 65（推荐本周发）

2. 「海外客户开会的 5 个坑」
   爆款 60 / 人设 80（推荐下周发）

3. 「为什么 90% 的人学 AI 坚持不下去」
   爆款 80 / 人设 75（推荐随时发）

🔥 最值得发：「AI 周报自动化」
理由：实用价值高，能真正节省创作者时间

📊 看板已更新：~/.media-topic-skill/preview.html
```

**然后你可以：**
- 打开看板文件（路径在上方输出中）查看可视化看板
- 继续输入更多灵感
- 询问 AI 哪个选题最值得发

```

```

## 🎯 评分系统说明

每个选题会从 4 个维度评分（0-100）：

| 维度 | 说明 | 评分标准 |
|------|------|----------|
| 🔥 爆款指数 | 有多少人想看？ | 话题热度、标题吸引力、情绪触发、时效性 |
| 👤 人设价值 | 对个人品牌有多大帮助？ | 独特性、经历绑定、差异化、长期价值 |
| 💰 商业价值 | 能带来多少商业机会？ | 变现路径、受众购买力、合作可能性 |
| ⚔️ 竞争度 | 这个赛道有多卷？ | 内容供给、头部垄断、差异化空间（越低越好） |

**综合分 = 爆款×0.4 + 人设×0.3 + (100-竞争)×0.3**

## 📊 看板功能

打开看板文件（数据目录下的 `preview.html`），你会看到：

1. **✍️ 灵感收件箱** — 最近输入的灵感
2. **🤖 AI 内容总监** — AI 的内容策略建议
3. **🔥 近期最值得发** — Top 3 推荐选题
4. **📊 内容生态分析** — 主题分布是否均衡
5. **📋 选题库** — 所有选题，按主题分组
6. **📚 内容系列** — 自动识别的内容系列
7. **📈 创作进度** — 统计数据
8. **💎 创作资产** — 累计数据
9. **👤 创作者画像** — 你的身份信息
10. **🪦 暂不推荐** — 低分或不适合的选题
11. **🎯 选题价值地图** — 可视化分布

## 💡 使用技巧

### 1. 批量输入效率更高

一次输入 5-10 个想法，比一个一个输入效率高。

### 2. 不要担心格式

AI 会自动识别和整理，你只需要表达想法。

### 3. 定期回顾看板

每周打开 `preview.html` 看看，了解你的内容生态。

### 4. 善用"系列"功能

如果多个想法有关联，AI 会自动识别为系列。

### 5. 及时标记已发布

发布后告诉 AI，它会更新状态并推荐下一个。

## ❓ 常见问题

### Q: 我需要懂技术吗？

**不需要。** 你只需要会复制粘贴。

### Q: 我的数据安全吗？

**完全安全。** 所有数据都在你的电脑上，不会上传到任何服务器。

### Q: 支持哪些平台？

所有主流平台：小红书、YouTube、B站、抖音、公众号、知乎、X/Twitter 等。

### Q: 可以多人协作吗？

目前只支持单人使用。未来可能会加入团队协作功能。

### Q: 如何更新版本？

下载最新版本，覆盖旧文件即可。你的数据不会丢失。

## 🆘 遇到问题？

### 问题 1：AI 没有自动处理

**解决：** 确保你已经告诉 AI 你是谁（首次使用）。

### 问题 2：看板没有更新

**解决：** 让 AI "帮我更新看板" 或 "重新生成 preview.html"。

### 问题 3：选题评分不合理

**解决：** 告诉 AI "我觉得 #X 的评分太高/太低了，因为..."，AI 会调整。

### 问题 4：想删除某个选题

**解决：** 告诉 AI "帮我删除 #X 号选题"。
