# Creator OS 快速开始

## 30 秒快速体验

### 第 1 步：下载项目

点击 GitHub 页面的 `Code` → `Download ZIP`，解压到桌面。

### 第 2 步：打开 AI 助手

打开 Claude Code、Cursor 或其他 AI 助手。

### 第 3 步：输入灵感

直接输入：

```
我是一个 AI 职场成长创作者，主平台是小红书和 YouTube，目标是建立个人 IP。

今天想到：
* Claude Code 比 Cursor 好用
* AI 周报自动化能节省 2 小时
* 海外客户开会要确认时区
```

### 第 4 步：查看结果

AI 会自动回复：

```
✅ 已为你创建创作者画像！

📥 本次处理 3 个选题：

1. 「Claude Code vs Cursor：为什么我选择 Claude Code」
   爆款 85 / 人设 70 / 商业 60 / 竞争 45
   推荐：本周发布

2. 「AI 周报自动化：从 2 小时到 15 分钟」
   爆款 75 / 人设 65 / 商业 70 / 竞争 50
   推荐：下周发布

3. 「海外客户开会的 5 个坑」
   爆款 60 / 人设 80 / 商业 55 / 竞争 40
   推荐：随时可发

🔥 最值得发：「Claude Code vs Cursor」
理由：AI 工具类内容热度高，且你有真实使用经验

📊 看板已更新：~/.media-topic-skill/preview.html
```

**完成！** 你现在有了一个完整的选题库和内容看板。

---

## 日常使用

### 录入灵感

随时把想法告诉 AI：

```
今天想到：
* 为什么学 AI 的人坚持不下去
* 理工科转运营的真实经历
```

### 查看看板

打开数据目录下的 `preview.html` 文件，查看你的内容看板。

### 查看选题库

告诉 AI：
```
看看我的选题库
```

AI 会列出所有选题。

### 标记已发布

告诉 AI：
```
标记 #1 已发布
```

AI 会帮你标记。

---

## 常用命令

| 你说 | AI 做什么 |
|------|----------|
| "看看我的选题库" | 列出所有选题 |
| "帮我更新 #3" | 修改选题信息 |
| "标记 #1 已发布" | 标记为已发布 |
| "把 #2 和 #5 合并" | 合并重复选题 |
| "帮我看看哪个该发了" | 推荐 Top 3 选题 |

---

## 看板功能

打开看板文件（数据目录下的 `preview.html`），你会看到：

- ✍️ **灵感收件箱** — 最近录入的灵感
- 🤖 **AI 内容总监** — AI 的内容策略建议
- 🔥 **近期最值得发** — 按综合分排序的 Top 3 选题
- 📊 **内容生态分析** — 主题分布和平衡性分析
- 📋 **选题库** — 所有选题，按主题分组
- 🎯 **选题价值地图** — 选题的可视化分布

---

## 支持的语言

- **界面语言**：中文 / English（点击右上角切换）
- **内容语言**：保留原文（默认）

---

## 数据位置

数据自动保存在可写位置（优先主目录，不可写时自动切换到项目内 `data/`）：
- 默认：`C:\Users\你的用户名\.media-topic-skill\`（Windows）
- 备选：项目内的 `data/` 目录

**备份建议：** 定期复制数据文件夹。
